Rubicon Python SDK

**rubi-py** یک SDK پایتون برای تعامل با پروتکل کتاب سفارش Rubicon است. شما می توانید با پایتون از دسترسی به تعویض توکن، سفارش محدود، سفارشات بازار و سفارشات دسته ای استفاده کنید!

این SDK یک نرم افزار منبع باز است که تحت مجوز Apache 2.0 صادر شده است. می توانید یک موضوع را باز کنید و در [codebase](https://github.com/RubiconDeFi/rubi-py) مشارکت کنید در GitHub، در اینجا تعدادی خوب وجود دارد [first issues](https://github.com/RubiconDeFi/rubi-py/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22).
