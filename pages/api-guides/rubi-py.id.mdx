Rubicon Python SDK

**rubi-py** adalah Python SDK untuk berinteraksi dengan protokol buku pesanan Rubicon. Anda dapat menggunakan akses Token Swap, Limit Order, Market Order, dan Batch Order dengan Python!

SDK ini adalah perangkat lunak sumber terbuka yang diterbitkan di bawah lisensi Apache 2.0. Anda dapat membuka masalah dan berkontribusi pada [codebase](https://github.com/RubiconDeFi/rubi-py) di GitHub, ini beberapa yang bagus [first issues](https://github.com/RubiconDeFi/rubi-py/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22).
