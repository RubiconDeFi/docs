# SDK de Python

`rubi-py` es un SDK para interactuar con el protocolo Rubicon usando Python.

## Descargo de responsabilidad
Este código base está en Alpha, podría contener problemas y puede cambiar significativamente entre versiones.

## Instalación
Instale el paquete rubi:

```
pip install rubi
```

o

```
poetry add rubi
```
