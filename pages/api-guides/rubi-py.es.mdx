SDK de Python Rubicon

**rubi-py** es un SDK de Python para interactuar con el protocolo de libro de pedidos de Rubicon. ¡Puede usar tokens de acceso, órdenes limitadas, órdenes de mercado y órdenes por lotes con Python!

Este SDK es un software de código abierto emitido bajo una licencia Apache 2.0. Puede abrir una edición y contribuir al [codebase](https://github.com/RubiconDeFi/rubi-py) en GitHub, aquí hay algunas buenas [first issues](https://github.com/RubiconDeFi/rubi-py/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22).
