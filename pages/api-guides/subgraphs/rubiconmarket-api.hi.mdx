# रूबिकनमार्केट एपीआई

## **रुबिकॉन मार्केट की प्रमुख संस्थाएं**

RubiconMarket सबग्राफ में संस्थाओं का एक संग्रह होता है जो प्रोटोकॉल की ऑर्डर बुक पर ट्रेडिंग गतिविधि से संबंधित डेटा एकत्र और संग्रहीत करता है। नीचे प्रमुख इकाइयां हैं जो हमें विश्वास है कि उपयोगकर्ता अपनी डेटा आवश्यकताओं के लिए सबसे अधिक प्रासंगिक पाएंगे।

### _**उपयोगकर्ता इकाई (आईडी: वॉलेट पता)**_

``` ग्राफक्यूएल
उपयोगकर्ता @entity टाइप करें {
    "<user address>"
    मैंने किया!
    "व्युत्पन्न मान"
    स्वैप: [स्वैप!]! @derivedFrom (फ़ील्ड: "उपयोगकर्ता")
    ओपनऑफर: [ओपनऑफर!]! @derivedFrom (फ़ील्ड: "उपयोगकर्ता")
    ऑफ़र: [ऑफ़र!]! @derivedFrom (फ़ील्ड: "उपयोगकर्ता")
    लेने वाले: [लेने वाला!]! @derivedFrom (फ़ील्ड: "उपयोगकर्ता")
}
```

उपयोगकर्ता इकाई वॉलेट के पते से संबंधित सभी व्यापारिक गतिविधियों को एकत्र करने के लिए संदर्भ के एक बिंदु के रूप में कार्य करती है। एक उपयोगकर्ता इकाई के भीतर से आप स्वैप, ओपनऑफर, ऑफर और टेकर संस्थाओं तक पहुंच सकते हैं।
