روبيكون بايثون SDK

** rubi-py ** عبارة عن Python SDK للتفاعل مع بروتوكول دفتر أوامر روبيكون. يمكنك استخدام الوصول إلى مقايضات الرموز وأوامر الحد وأوامر السوق والطلبات المجمعة مع بايثون!

SDK هذا هو برنامج مفتوح المصدر تم إصداره بموجب ترخيص Apache 2.0. يمكنك فتح مشكلة والمساهمة في [codebase](https://github.com/RubiconDeFi/rubi-py) على GitHub ، وهنا بعض جيد [first issues](https://github.com/RubiconDeFi/rubi-py/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22).
