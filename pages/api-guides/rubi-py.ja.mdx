ルビコン Python SDK

**rubi-py** は、Rubicon オーダー ブック プロトコルとやり取りするための Python SDK です。アクセス トークン スワップ、指値注文、成行注文、バッチ注文を Python で使用できます。

この SDK は、Apache 2.0 ライセンスの下で発行されたオープンソース ソフトウェアです。問題を開いて [codebase](https://github.com/RubiconDeFi/rubi-py) に貢献できます。 GitHub で、ここにいくつかの良い [first issues](https://github.com/RubiconDeFi/rubi-py/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22) があります。
