Рубикон Python SDK

**rubi-py** — это Python SDK для взаимодействия с протоколом стакана заказов Rubicon. Вы можете использовать свопы токенов доступа, лимитные ордера, рыночные ордера и пакетные ордера с Python!

Этот SDK представляет собой программное обеспечение с открытым исходным кодом, выпущенное под лицензией Apache 2.0. Вы можете открыть вопрос и внести свой вклад в [codebase](https://github.com/RubiconDeFi/rubi-py) на GitHub, вот несколько хороших [first issues](https://github.com/RubiconDeFi/rubi-py/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22).
