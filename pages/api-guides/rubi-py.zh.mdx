Rubicon Python SDK

**rubi-py** 是一个用于与 Rubicon 订单簿协议交互的 Python SDK。您可以使用 Python 访问代币互换、限价订单、市价订单和批量订单！

此 SDK 是根据 Apache 2.0 许可发布的开源软件。您可以打开一个问题并为 [codebase](https://github.com/RubiconDeFi/rubi-py) 做出贡献在 GitHub 上，这里有一些不错的 [first issues](https://github.com/RubiconDeFi/rubi-py/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22)。
