Rubicon Python SDK

**rubi-py** est un SDK Python pour interagir avec le protocole de carnet de commandes Rubicon. Vous pouvez accéder aux Token Swaps, Limit Orders, Market Orders et Batch Orders avec Python !

Ce SDK est un logiciel open source publié sous une licence Apache 2.0. Vous pouvez ouvrir un sujet et contribuer au [codebase](https://github.com/RubiconDeFi/rubi-py) sur GitHub, voici du bon [first issues](https://github.com/RubiconDeFi/rubi-py/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22).
