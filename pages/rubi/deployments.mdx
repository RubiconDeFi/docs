# Protocol Deployments

Rubicon is currently deployed and active on multiple chains, providing decentralized trading infrastructure across the Ethereum ecosystem. Governance takes place on L1 Ethereum.

## Active Networks

### Ethereum
- Primary governance network for the protocol
- Core governance execution and decision-making layer
- Home of the original $RUBI token
- Full protocol deployment
