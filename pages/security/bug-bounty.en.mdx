# Bug Bounties

## Immunefi Bug Bounty Program

Rubicon has a bug bounty program covering the smart contracts with a maximum payout of $50,000. Visit the [Rubicon Bug Bounty page](https://immunefi.com/bounty/rubicon/) on Immunefi to learn more. 

## Other Vulnerabilities

For vulnerabilities in Rubicon sites, frontends, servers, or other web properties that are not covered in the Immunefi program, please send a note to **security@rubicon.finance** with detailed instructions that confirm the vulnerability.
