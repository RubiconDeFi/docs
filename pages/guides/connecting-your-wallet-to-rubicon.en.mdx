# Connecting a Wallet ⛓️

Connecting to the Rubicon App for the first time using Mobile or Desktop/Browser wallets. If you're familiar with [Ethereum wallets](https://ethereum.org/en/wallets/) and how to add custom networks, you can skip this guide!

## Mobile Wallet

This guide uses the [MetaMask Mobile App](https://metamask.io/), which is available on Android and iOS.

1.  On the Browser tab, search for app.rubicon.finance. Make sure the URL is **app.rubicon.finance**!!! You can bookmark it by tapping on the **3 dots** in the bottom left and **Add to Favorites**.

    ![](</assets/MMBrowser.png>)

2.  The Rubicon App will ask you to connect your wallet. Tap on the **MetaMask** button:

    ![](</assets/RBKit_WalletFlow.png>)

3.  Your wallet will then ask you to connect to the Rubicon App. Tap **Connect**:

    ![](</assets/RBKit_ConnectFlow.png>)

Once connected, you can start swapping and trading on Rubicon!

## Desktop/Browser Wallet

This guide will use the [MetaMask browser extension](https://metamask.io/) wallet.

Let's start by connecting our wallet to Rubicon. Launch the Rubicon App by visiting [app.rubicon.finance](https://app.rubicon.finance/). You will see this wallet connection screen:

![](</assets/RBKit_WebFlow.png>)

Click on the MetaMask button, and the Rubicon App will ask for permission to connect to your wallet. Click **Connect**:

![](</assets/MMConnectRequest.png>)

Now your wallet is connected to Rubicon! Once you are connected to the Rubicon app, you will be prompted to add the Optimism (or Arbitrum) network to your wallet:

![](</assets/MMAddOPMainnet.png>)

## Adding a Custom Network

If for some reason you cannot automatically add these networks, you might need to do it manually. Here are the steps to add a custom network on MetaMask:

1.  Open MetaMask and tap on the **Menu icon** in the top left:

    ![](</assets/MMTopBar.png>)

2.  Tap **Settings** in the menu:

    ![](</assets/MMSettings.png>)

3.  In the Settings menu, select **Networks** to add a custom network:

    ![](</assets/MMCustomNetworks.png>)

4.  Tap on the **Add Network** button at the bottom of the screen:

    ![](</assets/MMAddNetwork.png>)

5.  Here are the RPC network details for Arbitrum One and Optimism Mainnet:

    | Network            | Arbitrum One |
    | :----------------- | :------- |
    | RPC Url            | https://arb1.arbitrum.io/rpc |
    | Chain ID           | 42161 |
    | Symbol             | ETH |
    | Block Explorer URL | [https://arbiscan.io/](https://arbiscan.io/) |

    | Network            | Optimism Mainnet |
    | :----------------- | :------- |
    | RPC Url            | https://mainnet.optimism.io |
    | Chain ID           | 10 |
    | Symbol             | ETH |
    | Block Explorer URL | [https://optimistic.etherscan.io](https://optimistic.etherscan.io) |
