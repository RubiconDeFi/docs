# Passer à l'optimisme

Pour commencer à négocier ou à fournir des liquidités sur Rubicon, vous avez besoin d'ETH sur [Optimism](https://www.optimism.io/).

Utilisez le pont dans le [Rubicon App](https://app.rubicon.finance/) pour trouver le meilleur itinéraire de transition !

## Guide des ponts

Ce guide vous montre comment relier les jetons à l'Optimisme d'une autre chaîne. Notre exemple convertit USDC sur Polygon en ETH sur Optimism :

1. Connectez-vous à l'application Rubicon et sélectionnez **Bridge** dans la barre latérale :
   
   ![](</assets/image(96).png>)

2. Sélectionnez la chaîne depuis et vers laquelle vous transférez. Dans cet exemple, nous transférons de Polygon à Optimism :
   
   ![](/assets/image.png)

3. Sélectionnez le(s) jeton(s) que vous souhaitez envoyer et recevoir, ils ne doivent pas nécessairement être identiques ! Ici, nous transférons USDC sur Polygon pour ETH sur Optimisme :
   
   ![](</assets/image(115).png>)

4. Après avoir sélectionné le nombre de jetons que vous souhaitez transférer, vous serez invité à passer à la chaîne "Transfert de" dans votre portefeuille (si vous ne l'avez pas déjà fait) :
   
   ![](</assets/image(120).png>)
   
   ![](</assets/image(39).png>)

5. Le pont trouve le meilleur itinéraire qui vous rapporte le plus de jetons sur la chaîne de destination. Vous pouvez également choisir un itinéraire moins cher. Dans cet exemple, le pont Hyphen nous rapporte le plus d'ETH sur l'optimisme et a des frais peu élevés :
   
   ![](</assets/image(101).png>)

6. Vous devrez **Approuver** le contrat Socket Registry. Sélectionnez **Approuver** et confirmez la transaction d'approbation dans votre portefeuille :
   
   ![](</assets/image(6).png>)
   
   ![](</assets/image(5).png>)

7. Une fois approuvé, vous êtes prêt à faire le pont ! Sélectionnez **Bridge** et confirmez la transaction dans votre portefeuille :
   
   ![](</assets/image(85).png>)

8. Les jetons sont en route ! Cela peut prendre environ 10 minutes pour que les jetons apparaissent dans votre portefeuille sur la chaîne de destination. En attendant, vous pouvez explorer le reste de l'application Rubicon



Pour obtenir de l'aide avec une transaction de pont, visitez Socket's [Knowledge Base](https://socketdottech.zendesk.com/hc/en-us) pour obtenir des réponses à la FAQ et de l'aide pour le dépannage.
