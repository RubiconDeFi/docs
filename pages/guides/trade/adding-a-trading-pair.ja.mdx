# 取引ペアの追加

ルビコン プロトコルはパーミッションレスです。オーダー ブック コントラクトは、任意の [ERC-20](https://ethereum.org/en/developers/docs/standards/tokens/erc-20/) の取引をサポートしています。トークン！



[**Rubicon App**](https://app.rubicon.finance/trade) にローカルで新しいトークンを使用して取引ペアを追加できます。ほんの数ステップで:

1. **取引** ページの上部にある **トークン メニュー** を選択します。
   ![トークンメニュー](</assets/image(15).png>)

2. 新しい取引ペアに必要な **Quote Token** を選択します。
   ![引用トークン](</assets/image(106).png>)

3. **Add Token** をクリックし、新しいトークンのコントラクト アドレスを貼り付けます。 Optimism でトークンの正しいアドレスを追加してください。[Optimism block explorer](https://optimistic.etherscan.io/tokens) でこれを確認してください。
   ![トークンを追加](</assets/image(55).png>)

4. トークン アドレスを貼り付けて追加すると、メニューに新しい取引ペアが表示されます。
   ![新しいペア](</assets/image(62).png>)

5. 新しいペアを選択して新しいオーダーブックを表示し、指値注文を開始します。
   ![新しいペアを選択](</assets/image(33).png>)

新しいオーダーブックの流動性を構築することは困難な場合があります。新しい通貨ペアでは、ブックに指値注文を出して流動性を追加する最初の通貨ペアになる必要があるかもしれません

ルビコンのマーケット メーカーは新しいペアを監視し、勢いがついた場合にサポートを追加できます。チームは、誰でも新しい流動性プール、取引ペアを作成し、1 回のトランザクションでオーダー ブックを初期化できる、より恒久的なソリューションに取り組んでいます。
