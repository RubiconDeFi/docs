Use **[Swap](https://app.rubicon.finance/swap)** to find the best route for a token swap on the Rubicon order books. Swaps pay the [Taker Fee](/protocol/rubicon-market/fees).

![](</assets/SwapExample.png>)

## Slippage

Use the settings icon to adjust the slippage tolerance for your swap. The transaction will revert if the price changes by more than this percentage. The default is 0.2%. 

![](</assets/Slippage.png>)
