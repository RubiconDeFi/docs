# Ikhtisar Perdagangan

Di halaman **[Trade](https://app.rubicon.finance/trade)**, perdagangkan pasangan token ERC-20 dengan Limit Order dan Market Order di Aplikasi Rubicon!

Grafik TradingView menampilkan data perdagangan Rubicon historis untuk setiap pasangan token.

![](</assets/Trade Page.png>)

## Jenis Pesanan

Market Order menentukan kuantitas dan mengeksekusi pada harga terbaik yang tersedia. Pesanan Pasar membayar [Taker Fee](/protocol/rubicon-market/fees).

Limit Order menentukan harga dan jumlah dan diisi jika harga pasar melewati batas harga. Limit Order menghasilkan [Maker Fee](/protocol/rubicon-market/fees) ketika mereka diisi.

![](</assets/MarketOrder.png>)
