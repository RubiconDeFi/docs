# Menambahkan Pasangan Perdagangan

Protokol Rubicon tidak memiliki izin: kontrak buku pesanan mendukung perdagangan apa pun [ERC-20](https://ethereum.org/en/developers/docs/standards/tokens/erc-20/) token!



Anda dapat menambahkan pasangan perdagangan dengan token baru secara lokal ke [**Rubicon App**](https://app.rubicon.finance/trade) hanya dalam beberapa langkah:

1. Pilih **Menu Token** di bagian atas halaman **Perdagangan**:
   ![Menu Token](</assets/image(15).png>)

2. Pilih **Token Kutipan** yang Anda inginkan untuk Pasangan Perdagangan baru:
   ![Token Kutipan](</assets/image(106).png>)

3. Klik **Tambah Token** dan tempelkan alamat kontrak token baru. Pastikan untuk menambahkan alamat yang benar untuk token di Optimism, verifikasi ini di [Optimism block explorer](https://optimistic.etherscan.io/tokens):
   ![Tambah Token](</assets/image(55).png>)

4. Setelah Anda menempelkan alamat token dan menambahkannya, Anda akan melihat pasangan perdagangan baru di menu Anda:
   ![Pasangan Baru](</assets/image(62).png>)

5. Pilih pasangan baru untuk melihat buku order baru dan mulai menempatkan limit order:
   ![Pilih Pasangan Baru](</assets/image(33).png>)

Membangun likuiditas untuk buku pesanan baru bisa jadi sulit. Pada pasangan baru, Anda mungkin harus menjadi yang pertama menambahkan likuiditas dengan menempatkan pesanan limit di buku

Pembuat pasar Rubicon memantau pasangan baru dan dapat menambahkan dukungan untuk pasangan jika mendapatkan daya tarik. Tim sedang mengerjakan solusi yang lebih permanen yang memungkinkan siapa saja membuat kumpulan likuiditas baru, pasangan perdagangan, dan menginisialisasi buku pesanan dalam satu transaksi.
