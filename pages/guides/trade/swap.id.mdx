[Swap](https://app.rubicon.finance/swap) temukan rute terbaik untuk pertukaran token di buku pesanan Rubicon:

![](</assets/image(18).png>)

Pilih token untuk mencari aset yang dapat diperdagangkan:

![](</assets/image(89).png>)

Jika buku pesanan tidak ada untuk token yang Anda tukarkan, router akan mengirimkan swap melalui token cair seperti (W)ETH atau USDC. Untuk berdagang langsung di buku pesanan, kunjungi [Trade](/docs/guides/trade/trade).
