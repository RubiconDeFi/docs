Gunakan **[Swap](https://app.rubicon.finance/swap)** untuk menemukan rute terbaik untuk pertukaran token di buku pesanan Rubicon. Swap membayar [Taker Fee](/protocol/rubicon-market/fees).

![](</assets/SwapExample.png>)

## Slip

Gunakan ikon pengaturan untuk menyesuaikan toleransi selip untuk swap Anda. Transaksi akan dikembalikan jika harga berubah lebih dari persentase ini. Standarnya adalah 0,2%.

![](</assets/Slippage.png>)
