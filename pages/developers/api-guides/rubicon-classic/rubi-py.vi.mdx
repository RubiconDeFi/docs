# Python SDK  

`rubi-py` là SDK để tương tác với giao thức Rubicon bằng Python.

## Từ chối trách nhiệm
Cơ sở mã này đang ở giai đoạn Alpha, nó có thể chứa các sự cố và có thể thay đổi đáng kể giữa các phiên bản.

## Cài đặt
Cài đặt rubi:

```
pip install rubi
```

or 

```
poetry add rubi
```
