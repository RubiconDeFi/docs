# Python SDK  

`rubi-py` is an SDK for interacting with the Rubicon protocol using Python.

## Disclaimer
This codebase is in Alpha, it could contain issues and may change significantly between versions.

## Installation
Install the rubi package:

```
pip install rubi
```

or 

```
poetry add rubi
```
